## 📖 What is ctxpack?

ctxpack is a simple tool designed to help you manage and compress large blocks of text. If you work with text that takes up too much space—like summaries, notes, or chunks of information used in AI language models—ctxpack can help shrink that text. The smaller size makes it easier to store, share, or use in software that limits how much text it can handle at once.

You don’t need to be a programmer or have technical skills to use ctxpack. This guide will walk you through downloading and running the software, step by step.

---

## 🔎 Why Use ctxpack?

- Compress text data to save storage space.
- Manage text in a way that fits model limits for AI tools.
- Easily compress and decompress text without complex tools.
- Work with text files on your computer without programming.

## 📝 How to Use ctxpack

Once ctxpack is running, you will see an easy interface that lets you compress or decompress text files.

- **To Compress:**  
  Click "Open File" or drag your text file into the app. Then press "Compress." ctxpack will shrink the text and save a new, smaller file.

- **To Decompress:**  
  Open one of your compressed files in ctxpack and choose "Decompress" to restore the original text.

### Supported File Types

- Plain text (`.txt`)
- JSON files containing text
- Other common text formats

### Saving Files

After processing, the app will prompt you where to save the new file. Choose a folder on your computer that you will remember.
